# Requirements

In this section we will go into detail about the required softwares, hardware and tools in order to build one speaker. Multiply each quantity by the number of speaker you want :no_mouth:.

You can download all the necessary files via [this link](../assets/dotpi-box/requirements/dotpi-box-files-V1.zip)


## Softwares

Here are a list of softwares you will need in order to carry out the project
- 3D slicer (e.g. [Bambu Studio](https://bambulab.com/fr-fr/download/studio), depending on the 3D printer you use)
- Vector Graphics Editor (e.g. [Inkscape](https://inkscape.fr/))
- Optionally Fusion 360 if you want to edit the project (see [Modify the project](modify-the-project.md) if that is the case)

## Hardware

You will need access to these machines in order to craft pieces :

- A 3D printer
- A laser cutting machine
